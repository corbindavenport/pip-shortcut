Keyboard shortcut

The current keyboard shortcut for Picture-in-Picture mode is currently and can be changed from your browser's settings.

Open about:addons in a new tab, click the settings gear button, then select "Manage extension shortcuts."

How to use this extension

Picture-in-Picture Shortcut adds a keyboard shortcut and toolbar button for switching video players to the Picture-in-Picture mode in your web browser. The Picture-in-Picture Shortcut extension button in the browser's toolbar can be used to toggle PiP mode.

Picture-in-Picture Shortcut also overrides sites that don't normally allow Picture-in-Picture mode, and attempts to add a play/pause button to the PiP popup. However, this may not always work for sites and players not originally built for PiP mode.
